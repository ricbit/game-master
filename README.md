# Konami's Game Master

Complete disassembly of the MSX cartridge "Konami's Game Master"
(RC-735), a 1985 utility cart that adds save/load, cheats, and a
ranking system to other Konami MSX1 game cartridges.

This source produces a bit-perfect binary of the original 16 KB ROM. The
MD5 of the assembled output matches `9d5784c6b9b0f1dd3530b36edcdcc47f`.

Compile using [sjasmplus](https://github.com/z00m128/sjasmplus).
